HTML Basics Summary

HTML, which stands for HyperText Markup Language, is the foundation of web pages, defining their structure and content. It uses elements and tags to specify how content like text, images, and links should be displayed in a browser. HTML provides the basic framework for websites, organizing information into headings, paragraphs, lists, and more.

CSS Basics Summary

Cascading Style Sheets (CSS) is a stylesheet language used to describe the presentation of a document written in HTML or XML (including XML dialects such as SVG, MathML or XHTML). CSS describes how elements should be rendered on screen, on paper, in speech, or on other media.

JavaScript Basics Summary

JavaScript is a dynamic, interpreted programming language primarily used to add interactivity to websites. It's lightweight, cross-platform, and runs directly in web browsers without needing compilation. JavaScript manipulates the Document Object Model (DOM) to modify HTML and CSS, enabling features like form validation, animations, and dynamic content updates. It supports object-oriented programming concepts and uses a syntax similar to Java and C. While often associated with client-side web development, JavaScript can also be used server-side with Node.js.

How To Make A Website Attractive

  1. Keep your design balanced.
  2. Compartmentalize your design by using grids
  3. Pick two or three base colors at most for your design.
  4. Try to make the graphics go well together.
  5. Improve your website's typography.
  6. Make elements stand out by adding white space around them.

Bonus Point On web development

In web development, a "bonus point" typically refers to a feature or aspect that provides an added benefit, especially when it comes to improving the user experience or the website's functionality. These can range from technical skills like understanding SEO to design choices like using a personal headshot in a portfolio.
